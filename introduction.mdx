Spoo.me is a powerful URL shortening service that provides a comprehensive API for shortening URLs, managing analytics, and exporting data. Our API is designed to be fast, reliable, and easy to integrate into your applications.

<Info>
**New API Documentation!** We've recently moved to this updated documentation. If you need to access the previous version, [click here to view the old docs](https://spoo.me/api?old=1).
</Info>

![Spoo.me Banner](/images/banner-rounded.png)

## Base URL

**API v1 (Recommended)**
```
https://spoo.me/api/v1
```

**API v0 (Legacy)**
```
https://spoo.me
```
