This endpoint allows you to export comprehensive data of your shortened URL in multiple formats. The response contains raw data in the specified format.

## Endpoint

<CodeGroup>
```http Request
GET https://spoo.me/export/{shortCode}/{exportFormat}
```
</CodeGroup>

<Note>
`shortCode` can represent the short code of both regular and emoji URLs.
</Note>

## Path Parameters

<Fields>
  <Field name="shortCode" type="string" required>
    The short code of the URL you want to export data for
  </Field>
  <Field name="exportFormat" type="string" required>
    The format for data export. Available options: `json`, `csv`, `xlsx`, `xml`
  </Field>
</Fields>

## Headers

| Header | Type | Required | Value |
|--------|------|----------|-------------|
| Content-Type | string | Yes | `application/x-www-form-urlencoded` |

## Request Body

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| password | string | No | Password of the shortened URL if it's password protected. Required only for password-protected URLs. |

<Warning>
**Important Note**: The API returns raw data in the specified format, which you need to store in a file. For CSV format, ensure you save the file with a `.zip` extension as the API returns **multiple CSV files bundled together**.
</Warning>

## Code Examples

<CodeGroup>
```python Python
import requests

# Replace these with your actual values
short_code = "exa"
base_url = "https://spoo.me"
export_format = "xlsx"    # You can choose from ["json", "csv", "xlsx", "xml"]

payload = {                      # not required if your short url is not password protected
    "password": "Example@12"
}

# Make the request
response = requests.post(
    f"{base_url}/export/{short_code}/{export_format}",
    data=payload
)

# Check the response
if response.status_code == 200:
    # If the request was successful, store the data in a file
    file_extension = export_format if export_format != "csv" else "zip"
    with open(f"{short_code}.{file_extension}", "wb") as file:
        file.write(response.content)
    print(f"Data exported successfully to {short_code}.{file_extension}")
else:
    # If the request failed, print the error message
    print(f"Error: {response.status_code}")
    print(response.text)
```

```javascript Node.js
const axios = require('axios');
const fs = require('fs');

.catch(error => {
    console.error('Error:', error.response?.status);
    console.error(error.response?.data?.toString());
```

```javascript JavaScript (Fetch)
const shortCode = 'exa';
const baseUrl = 'https://spoo.me';

const payload = new URLSearchParams({

    const a = document.createElement('a');

    a.href = url;
    a.download = `${shortCode}.${fileExtension}`;
    document.body.appendChild(a);
    a.click();
    window.URL.revokeObjectURL(url);
    console.log(`Data exported successfully`);
})
.catch(error => {
    console.error('Error:', error);
```

```csharp C#
using System;
using System.Collections.Generic;
using System.IO;
using System.Net.Http;
using System.Threading.Tasks;

class Program
{
    static async Task Main(string[] args)
    {
        using (var client = new HttpClient())
        {
            var shortCode = "exa";
            var baseUrl = "https://spoo.me";
            var exportFormat = "xlsx"; // You can choose from ["json", "csv", "xlsx", "xml"]
            
            var payload = new List<KeyValuePair<string, string>>
            {
                new KeyValuePair<string, string>("password", "Example@12")
            };
            
            var response = await client.PostAsync($"{baseUrl}/export/{shortCode}/{exportFormat}", 
                new FormUrlEncodedContent(payload));
            
            if (response.IsSuccessStatusCode)
            {
                var fileExtension = exportFormat != "csv" ? exportFormat : "zip";
                var fileName = $"{shortCode}.{fileExtension}";
                var content = await response.Content.ReadAsByteArrayAsync();
                
                await File.WriteAllBytesAsync(fileName, content);
                Console.WriteLine($"Data exported successfully to {fileName}");
            }
            else
            {
                Console.WriteLine($"Error: {response.StatusCode}");
                var content = await response.Content.ReadAsStringAsync();
                Console.WriteLine(content);
            }
        }
    }
}
```

```java Java
import java.io.*;
import java.net.HttpURLConnection;
import java.net.URL;
import java.net.URLEncoder;
import java.nio.charset.StandardCharsets;
import java.nio.file.Files;
import java.nio.file.Paths;

public class SpooMeExport {
    public static void main(String[] args) throws IOException {
        String shortCode = "exa";
        String baseUrl = "https://spoo.me";
        String exportFormat = "xlsx"; // You can choose from ["json", "csv", "xlsx", "xml"]
        String urlString = baseUrl + "/export/" + shortCode + "/" + exportFormat;
        URL url = new URL(urlString);
        
        String payload = "password=" + URLEncoder.encode("Example@12", StandardCharsets.UTF_8);
        
        HttpURLConnection connection = (HttpURLConnection) url.openConnection();
        connection.setRequestMethod("POST");
        connection.setRequestProperty("Content-Type", "application/x-www-form-urlencoded");
        connection.setDoOutput(true);
        
        try (OutputStream os = connection.getOutputStream()) {
            byte[] input = payload.getBytes(StandardCharsets.UTF_8);
            os.write(input, 0, input.length);
        }
        
        if (connection.getResponseCode() == 200) {
            String fileExtension = !exportFormat.equals("csv") ? exportFormat : "zip";
            String fileName = shortCode + "." + fileExtension;
            
            try (InputStream inputStream = connection.getInputStream()) {
                Files.copy(inputStream, Paths.get(fileName));
                System.out.println("Data exported successfully to " + fileName);
            }
        } else {
            System.out.println("Error: " + connection.getResponseCode());
            try (BufferedReader br = new BufferedReader(new InputStreamReader(connection.getErrorStream(), StandardCharsets.UTF_8))) {
                StringBuilder response = new StringBuilder();
                String responseLine;
                while ((responseLine = br.readLine()) != null) {
                    response.append(responseLine.trim());
                }
                System.out.println(response.toString());
            }
        }
    }
}
```

```go Go
package main

import (
    "fmt"
    "io"
    "net/http"
    "net/url"
    "os"
    "strings"
)

func main() {
    shortCode := "exa"
    baseURL := "https://spoo.me"
    exportFormat := "xlsx" // You can choose from ["json", "csv", "xlsx", "xml"]
    apiURL := fmt.Sprintf("%s/export/%s/%s", baseURL, shortCode, exportFormat)
    
    data := url.Values{}
    data.Set("password", "Example@12")
    
    req, err := http.NewRequest("POST", apiURL, strings.NewReader(data.Encode()))
    if err != nil {
        fmt.Printf("Error creating request: %v\n", err)
        return
    }
    
    req.Header.Set("Content-Type", "application/x-www-form-urlencoded")
    
    client := &http.Client{}
    resp, err := client.Do(req)
    if err != nil {
        fmt.Printf("Error making request: %v\n", err)
        return
    }
    defer resp.Body.Close()
    
    if resp.StatusCode == 200 {
        fileExtension := exportFormat
        if exportFormat == "csv" {
            fileExtension = "zip"
        }
        fileName := fmt.Sprintf("%s.%s", shortCode, fileExtension)
        
        file, err := os.Create(fileName)
        if err != nil {
            fmt.Printf("Error creating file: %v\n", err)
            return
        }
        defer file.Close()
        
        _, err = io.Copy(file, resp.Body)
        if err != nil {
            fmt.Printf("Error writing file: %v\n", err)
            return
        }
        
        fmt.Printf("Data exported successfully to %s\n", fileName)
    } else {
        fmt.Printf("Error: %d\n", resp.StatusCode)
        body, _ := io.ReadAll(resp.Body)
        fmt.Printf("%s\n", string(body))
    }
}
```

## Response

The API returns raw binary data in the specified format. You need to save this data to a file with the appropriate extension.

<Note>
**CSV Format Special Note**: When exporting in CSV format, the API returns a ZIP file containing multiple CSV files with different data categories. Always save CSV exports with a `.zip` extension.
</Note>

## Error Responses

<Accordion title="UrlError (404)">
The requested URL (short code) never existed.

```json
{
  "error": "UrlError",
  "message": "The requested URL never existed"
}
```
</Accordion>

<Accordion title="PasswordError (400)">
The entered password is invalid for the password-protected URL.

```json
{
  "error": "PasswordError",
  "message": "Invalid password"
}
```
</Accordion>

<Accordion title="FormatError (400)">
The entered export format is invalid. Must be one of: `json`, `csv`, `xlsx`, `xml`.

```json
{
  "error": "FormatError",
  "message": "Invalid export format"
}
```
</Accordion>

<Warning>
**CSV Export Important**: If you're exporting data in CSV format, ensure you save the file with a `.zip` extension, not `.csv`. The API returns the data divided into multiple files compressed together.
</Warning>
